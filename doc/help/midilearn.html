MIDI Learn

In order to access the MIDI learn window you must first check the box for "MIDI Learn" in Settings/Preferences/MIDI of the MIDI Implementation section.

The MIDI Learn window allows you to create a MIDI implementation table for each preset in a bank. There are two browsers, one at the left with all the available effect parameters, and another at the right with all the presets in the Bank.

You can open this window clicking the right mouse over an effect parameter that can be controllable by MIDI. Also you can auto assign the incoming MIDI Control message "automatically" if you enable in Settings/Preferences/MIDI - Auto Assign MIDI Learn.

The radio buttons "All" and "Current" select the parameters listed in the browser. "Current" displays only the actual preset parameters available in MIDI learn.

The most important thing here is the input box located below the parameter browser. The MIDI control message can be edited by selecting and dragging, or capturing an incoming MIDI message by clicking on the Get MIDI Message button. When the desired control number is selected, it must be assigned to the effect by clicking on the assign button. The MIDI control will then be assigned to the selected effect parameter on the browser.

You can "Assign" up to 20 effects parameters for a single control number message. Also you can use different control messages to manage only one effect parameter.

The "Assign to All" adds this single assignment to all the presets in the bank, preserving everything that you have previously assigned in all the presets. The "Copy to All" button that overwrites all the assignments.

The grid boxes in the middle display the engaged MIDI control messages to the selected effect parameter on the browser, the Assignment for a single effect parameter can be cleared with the button "Clear Assignments", also you can clear all the assignments for this preset with the button Clear Preset".

In the left part of the window you have the list of presets in the working bank, you can copy from or to the selected preset, that copy all the preset table of a single preset.

The "Copy to All" button copy the actual preset to all the presets in the Bank.

The midi learn implementation tables is saved on the preset file for a single preset, and in a separate file in the case of the Banks, the program read this bank preset file when read the normal bank file.
